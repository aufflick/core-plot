Method Documentation

- (id) initWithFrame: (CGRect)  newFrame
implementation

Initializes a newly allocated CPTTradingRangePlot object with the provided frame rectangle.

This is the designated initializer. The initialized layer will have the following properties:

  • plotStyle = CPTTradingRangePlotStyleOHLC
  • lineStyle = default line style
  • increaseLineStyle = nil
  • decreaseLineStyle = nil
  • increaseFill = solid white fill
  • decreaseFill = solid black fill
  • barWidth = 5.0
  • stickLength = 3.0
  • barCornerRadius = 0.0
  • labelField = CPTTradingRangePlotFieldClose
Parameters:
newFrameThe frame rectangle.
Returns:
The initialized CPTTradingRangePlot object.

Reimplemented from CPTPlot.

- (BOOL) pointingDeviceDownEvent: (CPTNativeEvent *)  event
atPoint: (CGPoint)  interactionPoint 
implementation

Informs the receiver that the user has touched the screen.

If this plot has a delegate that responds to the -tradingRangePlot:barWasSelectedAtRecordIndex: and/or -tradingRangePlot:barWasSelectedAtRecordIndex:withEvent: methods, the interactionPoint is compared with each bar in index order. The delegate method will be called and this method returns YES for the first index where the interactionPoint is inside a bar. This method returns NO if the interactionPoint is outside all of the bars.

Parameters:
eventThe OS event.
interactionPointThe coordinates of the interaction.
Returns:
Whether the event was handled or not.
