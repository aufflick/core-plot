Detailed Description

A layer drawn on top of the graph layer and behind all plot elements.

All graph elements, except for titles, legends, and other annotations attached directly to the graph itself are clipped to the plot area frame.

- (id) initWithFrame: (CGRect)  newFrame
implementation

Initializes a newly allocated CPTPlotAreaFrame object with the provided frame rectangle.

This is the designated initializer. The initialized layer will have the following properties:

  • plotArea = a new CPTPlotArea with the same frame rectangle
  • needsDisplayOnBoundsChange = YES
